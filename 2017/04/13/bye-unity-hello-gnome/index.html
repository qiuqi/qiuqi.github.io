bye unity, hello gnome!

作为一个ubuntu的重度使用者，噩耗莫过于听到居然又要重启炉灶放弃unity，回到gnome。好吧，日子过久了，谁不腻呢。

  1. ClassicMenu Indicator
  2. Gnome优化工具，貌似用处不大，把日期显示调整了一下
  3. 在firefox上启用gnome extensions插件
  4. sudo apt-get install chrome-gnome-shell 用来配合浏览器插件
  5. 插件安装：Places Status Indicator
  6. 插件安装：Panel OSD 通过tweak tool启动，可以定制系统消息弹出的位置
  7. 卸载某些插件以后，需要通过alt+F2，输入r，然后回车，来重新启动gnome shell.
  8. 插件安装：activities-configurator来控制gnome的activities的行为，貌似没什么大用。
  9. 插件安装：TopIcons Plus， 用来把状态通知栏移动到顶部右上角。 记录那么多，其实就是为了实现这个配置，缺省配置在右下角太恶心了，严重干扰我写代码。
  10. 安装插件：TaskBar， 用Activities总觉得太耗眼神，还是固定的切换器比较好。
  11. 锁屏：ctrl+alt+l
  12. 安装插件：AlternateTab 不要以群组的方式显示任务切换
  13. 安装插件：Extension Update Notifier 插件更新提醒
  14. 安装插件：Recent(Item)s 显示最近打开文件列表
  15. 安装插件：Clipboard Indicator
