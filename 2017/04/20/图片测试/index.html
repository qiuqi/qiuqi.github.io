图片测试

在hexo中增加对图片的支持

  1. 修改_config.xml， post_asset_folder:true
  2. 安装hexo-asset-image插件
    npm install https://github.com/CodeFalling/hexo-asset-image –save
  3. 把图片拷贝到新建立的md文件名的目录中去
  4. 显示图片
